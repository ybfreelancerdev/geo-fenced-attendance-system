# Smart Geo-Fenced Attendance Management System

A full-stack solution for tracking attendance using real-time GPS and geo-fencing technology. This system ensures users can only mark attendance when they are within a predefined location boundary.

Built with a React Native mobile application and a secure backend API, it is ideal for organizations, field teams, and remote workforce monitoring.

---

## Key Highlights

* Geo-fencing based attendance validation
* Cross-platform mobile app (React Native)
* Secure backend with authentication
* Attendance tracking & history
* Real-time location verification

---

## Tech Stack

- React Native

Backend:
- .NET Core

## Setup Instructions

### Mobile App
cd attendance-mobile  
npm install  
npx react-native run-android  

### Backend API
cd attendance-api  
npm install / dotnet restore  
npm start / dotnet run
